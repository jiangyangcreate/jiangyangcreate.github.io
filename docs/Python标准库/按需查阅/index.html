按需查找

这些标准库模块使用频率较低，或已被更流行的第三方库事实性替代。它们仍然是 Python 标准库的一部分，在特定场景下有用，按需查阅即可。

📄️ argparse

argparse 用于解析命令行参数，自动生成帮助信息和错误提示，是编写 CLI 工具的标准方式。

📄️ array

array 模块提供了紧凑的类型化数组，所有元素必须是同一基本类型。它是理解 numpy.ndarray 设计思想的原型——连续内存布局、类型约束、高效存储。

📄️ base64

base64 模块提供将二进制数据编码为可打印 ASCII 字符（以及反向解码）的功能，支持 Base64、Base32、Base16 和 Base85 等编码方案。

📄️ code

code 模块提供了在 Python 中实现交互式解释器（REPL）的基础设施，可用于构建自定义的交互式控制台。

📄️ colorsys

colorsys 模块提供 RGB 与其他颜色空间（HSV、HLS、YIQ）之间的双向转换。

📄️ compression

compression包包含规范的压缩模块，其中包含几种不同压缩算法的接口。

📄️ csv

csv 模块用于读写 CSV（逗号分隔值）格式的表格数据，支持多种方言和自定义分隔符。

📄️ datetime、zoneinfo、calendar

datetime 模块提供日期和时间的核心操作，zoneinfo 处理时区，calendar 提供日历计算功能。

📄️ difflib

difflib 提供序列比较工具，可计算文本相似度、生成差异报告（diff）和 HTML 对比页面。

📄️ doctest

doctest 通过执行文档字符串中的交互式示例来验证代码行为，兼顾文档与测试。

📄️ ensurepip

ensurepip 用于在离线环境下为 Python 引导安装 pip 包管理器。

📄️ enum

enum 模块

📄️ getpass

getpass 模块提供便携的密码输入和用户名获取功能，输入密码时不会在终端回显。

📄️ glob

glob 模块提供 Unix shell 风格的路径名模式匹配，支持通配符 *、? 和 []。

📄️ hashlib

hashlib 模块提供了常见的安全哈希算法接口，包括 SHA-256、SHA-3、MD5、BLAKE2 等，用于数据完整性校验和密码学场景。

📄️ heapq

heapq 模块实现了堆队列算法（优先队列），用最小堆维护一个始终能以 O(1) 获取最小元素的列表结构。

📄️ html

html 模块提供 HTML 转义/反转义工具函数，html.parser 子模块提供基于事件的 HTML 解析器。

📄️ keyword

keyword 模块用于检测字符串是否为 Python 关键字或软关键字，并提供所有关键字的完整列表。

📄️ locale

locale 模块提供国际化与本地化服务，让程序能根据用户所在地区的文化习惯来格式化数字、货币、日期等。

📄️ math

math 模块提供了常用的数学函数和常量，涵盖取整、数论、幂运算、对数、三角函数等，底层由 C 数学库实现，运算高效。

📄️ mmap

mmap 模块提供内存映射文件的支持，允许像操作内存一样操作文件内容，无需将整个文件读入内存。这是处理大文件的高效方式，也是 numpy、HuggingFace datasets 等库处理大规模数据的底层技术之一。

📄️ pdb

pdb 是 Python 自带的交互式源码调试器，支持设置（条件）断点、单步执行、查看栈帧、列出源码以及在任意栈帧上下文中求值表达式。

📄️ pprint

pprint 用于美化打印复杂的 Python 数据结构，让嵌套的字典、列表等以层次清晰的格式输出。

📄️ random

random 模块提供伪随机数生成器，支持整数、浮点数、序列操作以及多种概率分布，基于 Mersenne Twister 算法实现。

📄️ re

re 模块 （正则表达式）

📄️ secrets

secrets 模块用于生成密码学安全的随机数，适合管理密码、令牌、安全凭据等敏感数据。

📄️ shutil

shutil 模块提供高级的文件和目录操作，补充了 os 和 pathlib 的不足。

📄️ socket

socket 模块提供了底层网络编程接口（BSD 套接字），是 Python 中进行 TCP/UDP 网络通信的基础。

📄️ sqlite3

sqlite3 模块提供了轻量级的 SQLite 嵌入式数据库接口，遵循 DB-API 2.0 规范，无需安装独立数据库服务器。

📄️ tarfile

tarfile 模块用于读写 tar 归档文件，支持 gzip、bz2、xz 等压缩格式。

📄️ tempfile

tempfile 模块用于安全地创建临时文件和目录，支持自动清理，避免残留临时数据。

📄️ time

time 模块提供底层的时间访问与转换功能，包括获取时间戳、休眠、格式化以及性能计时。

📄️ timeit

timeit 模块用于精确测量小段 Python 代码的执行时间，自动规避常见的计时陷阱。

📄️ tomllib

tomllib 模块用于解析 TOML 格式的配置文件（Python 3.11+ 内置，只读不写）。

📄️ unittest

unittest 是 Python 内置的单元测试框架，支持测试自动化、setUp/tearDown、测试聚合和 mock 等能力。

📄️ urllib

urllib 是一个处理 URL 的标准库包，包含请求发送、异常处理、URL 解析和 robots.txt 解析四个子模块。

📄️ venv

venv 用于创建轻量级的虚拟环境，使每个项目拥有独立的包依赖，互不干扰。

📄️ warnings

warnings 用于向开发者发出非致命性警告信息，常见于弃用通知、兼容性提醒等场景。

📄️ wave

wave 模块提供对 WAV 音频文件的读写接口，仅支持未压缩的 PCM 编码格式。

📄️ webbrowser

webbrowser 模块提供了一个高层接口，用于调用系统默认浏览器打开指定 URL。

📄️ xml

xml.etree.ElementTree 模块提供了轻量高效的 XML 解析与生成 API，是 Python 处理 XML 的首选工具。

📄️ zipfile

zipfile 模块提供创建、读取、写入和解压 ZIP 归档文件的功能。
