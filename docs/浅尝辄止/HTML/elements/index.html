е…¶д»–ж Үзӯҫ

жң¬з« д»Ӣз»ҚдёҖдәӣжңҖж–°еј•е…Ҙж ҮеҮҶзҡ„ж ҮзӯҫгҖӮ

<dialog>вҖӢ

еҹәжң¬з”Ёжі•вҖӢ

<dialog>ж ҮзӯҫиЎЁзӨәдёҖдёӘеҸҜд»Ҙе…ій—ӯзҡ„еҜ№иҜқжЎҶгҖӮ

<dialog>
  Hello world
</dialog>

дёҠйқўе°ұжҳҜдёҖдёӘжңҖз®ҖеҚ•зҡ„еҜ№иҜқжЎҶгҖӮ

й»ҳи®Өжғ…еҶөдёӢпјҢеҜ№иҜқжЎҶжҳҜйҡҗи—Ҹзҡ„пјҢдёҚдјҡеңЁзҪ‘йЎөдёҠжҳҫзӨәгҖӮеҰӮжһңиҰҒи®©еҜ№иҜқжЎҶжҳҫзӨәпјҢеҝ…йЎ»еҠ дёҠopenеұһжҖ§гҖӮ

<dialog open>
  Hello world
</dialog>

дёҠйқўд»Јз ҒдјҡеңЁзҪ‘йЎөжҳҫзӨәдёҖдёӘж–№жЎҶпјҢеҶ…е®№жҳҜHello worldгҖӮ

<dialog>е…ғзҙ йҮҢйқўпјҢеҸҜд»Ҙж”ҫе…Ҙе…¶д»– HTML е…ғзҙ гҖӮ

<dialog open>
  <form method="dialog">
    <input type="text">
    <button type="submit" value="foo">жҸҗдәӨ</button>
  </form>
</dialog>

дёҠйқўзҡ„еҜ№иҜқжЎҶйҮҢйқўпјҢжңүдёҖдёӘиҫ“е…ҘжЎҶе’ҢжҸҗдәӨжҢүй’®гҖӮ

жіЁж„ҸпјҢдёҠдҫӢдёӯ<form>зҡ„methodеұһжҖ§и®ҫдёәdialogпјҢиҝҷж—¶зӮ№еҮ»жҸҗдәӨжҢүй’®пјҢеҜ№иҜқжЎҶе°ұдјҡж¶ҲеӨұгҖӮдҪҶжҳҜпјҢиЎЁеҚ•дёҚдјҡжҸҗдәӨеҲ°жңҚеҠЎеҷЁпјҢжөҸи§ҲеҷЁдјҡе°ҶиЎЁеҚ•е…ғзҙ зҡ„returnValueеұһжҖ§и®ҫдёә Submit жҢүй’®зҡ„valueеұһжҖ§пјҲдёҠдҫӢжҳҜfooпјүгҖӮ

JavaScript APIвҖӢ

<dialog>е…ғзҙ зҡ„ JavaScript API жҸҗдҫӣDialog.showModal()е’ҢDialog.close()дёӨдёӘж–№жі•пјҢз”ЁдәҺжү“ејҖ/е…ій—ӯеҜ№иҜқжЎҶгҖӮ

const modal = document.querySelector('dialog');

// еҜ№иҜқжЎҶжҳҫзӨәпјҢзӣёеҪ“дәҺеўһеҠ  open еұһжҖ§
modal.showModal();

// еҜ№иҜқжЎҶе…ій—ӯпјҢзӣёеҪ“дәҺз§»йҷӨ open еұһжҖ§
modal.close();

ејҖеҸ‘иҖ…еҸҜд»ҘжҸҗдҫӣе…ій—ӯжҢүй’®пјҢи®©е…¶и°ғз”ЁDialog.close()ж–№жі•пјҢе…ій—ӯеҜ№иҜқжЎҶгҖӮ

Dialog.close()ж–№жі•еҸҜд»ҘжҺҘеҸ—дёҖдёӘеӯ—з¬ҰдёІдҪңдёәеҸӮж•°пјҢз”ЁдәҺдј йҖ’дҝЎжҒҜгҖӮ<dialog>жҺҘеҸЈзҡ„returnValueеұһжҖ§еҸҜд»ҘиҜ»еҸ–иҝҷдёӘеӯ—з¬ҰдёІпјҢеҗҰеҲҷreturnValueеұһжҖ§зӯүдәҺжҸҗдәӨжҢүй’®зҡ„valueеұһжҖ§гҖӮ

modal.close('Accepted');
modal.returnValue // "Accepted"

Dialog.showModal()ж–№жі•е”Өиө·еҜ№иҜқжЎҶж—¶пјҢдјҡжңүдёҖдёӘйҖҸжҳҺеұӮпјҢйҳ»жӯўз”ЁжҲ·дёҺеҜ№иҜқжЎҶеӨ–йғЁзҡ„еҶ…е®№дә’еҠЁгҖӮCSS жҸҗдҫӣдәҶдёҖдёӘ Dialog е…ғзҙ зҡ„::backdropдјӘзұ»пјҢз”ЁдәҺйҖүдёӯиҝҷдёӘйҖҸжҳҺеұӮпјҢеӣ жӯӨеҸҜд»Ҙзј–еҶҷж ·ејҸи®©йҖҸжҳҺеұӮеҸҳеҫ—еҸҜи§ҒгҖӮ

dialog {
  padding: 0;
  border: 0;
  border-radius: 0.6rem;
  box-shadow: 0 0 1em black;
}

dialog::backdrop {
  /* make the backdrop a semi-transparent black */
  background-color: rgba(0, 0, 0, 0.4);
}

дёҠйқўд»Јз ҒдёҚд»…дёә<dialog>жҢҮе®ҡдәҶж ·ејҸпјҢиҝҳе°ҶеҜ№иҜқжЎҶзҡ„йҖҸжҳҺеұӮеҸҳжҲҗдәҶзҒ°иүІйҖҸжҳҺгҖӮ

<dialog>е…ғзҙ иҝҳжңүдёҖдёӘDialog.show()ж–№жі•пјҢд№ҹиғҪе”Өиө·еҜ№иҜқжЎҶпјҢдҪҶжҳҜжІЎжңүйҖҸжҳҺеұӮпјҢз”ЁжҲ·еҸҜд»ҘдёҺеҜ№иҜқжЎҶеӨ–йғЁзҡ„еҶ…е®№дә’еҠЁгҖӮ

дәӢд»¶вҖӢ

<dialog>е…ғзҙ жңүдёӨдёӘдәӢд»¶пјҢеҸҜд»Ҙзӣ‘еҗ¬гҖӮ

  • closeпјҡеҜ№иҜқжЎҶе…ій—ӯж—¶и§ҰеҸ‘
  • cancelпјҡз”ЁжҲ·жҢүдёӢescй”®е…ій—ӯеҜ№иҜқжЎҶж—¶и§ҰеҸ‘

еҰӮжһңеёҢжңӣз”ЁжҲ·зӮ№еҮ»йҖҸжҳҺеұӮпјҢе°ұе…ій—ӯеҜ№иҜқжЎҶпјҢеҸҜд»Ҙз”ЁдёӢйқўзҡ„д»Јз ҒгҖӮ

modal.addEventListener('click', (event) => {
  if (event.target === modal) {
    modal.close('cancelled');
  }
});

<details>пјҢ<summary>вҖӢ

еҹәжң¬з”Ёжі•вҖӢ

<details>ж Үзӯҫз”ЁжқҘжҠҳеҸ еҶ…е®№пјҢжөҸи§ҲеҷЁдјҡжҠҳеҸ жҳҫзӨәиҜҘж Үзӯҫзҡ„еҶ…е®№гҖӮ

<details>
иҝҷжҳҜдёҖж®өи§ЈйҮҠж–Үжң¬гҖӮ
</details>

дёҠйқўзҡ„д»Јз ҒеңЁжөҸи§ҲеҷЁйҮҢйқўпјҢдјҡжҠҳеҸ иө·жқҘпјҢжҳҫзӨәDetailsпјҢеүҚйқўжңүдёҖдёӘдёүи§’еҪўпјҢе°ұеғҸдёӢйқўиҝҷж ·гҖӮ

в–¶ Details

з”ЁжҲ·зӮ№еҮ»иҝҷж®өж–Үжң¬пјҢжҠҳеҸ зҡ„ж–Үжң¬е°ұдјҡеұ•ејҖпјҢжҳҫзӨәиҜҰз»ҶеҶ…е®№гҖӮ

в–ј Details
иҝҷжҳҜдёҖж®өи§ЈйҮҠж–Үжң¬гҖӮ

еҶҚзӮ№еҮ»дёҖдёӢпјҢеұ•ејҖзҡ„ж–Үжң¬еҸҲдјҡйҮҚж–°жҠҳеҸ иө·жқҘгҖӮ

<details>ж Үзӯҫзҡ„openеұһжҖ§пјҢз”ЁдәҺй»ҳи®Өжү“ејҖжҠҳеҸ гҖӮ

<details open>
иҝҷжҳҜдёҖж®өи§ЈйҮҠж–Үжң¬гҖӮ
</details>

дёҠйқўд»Јз Ғй»ҳи®Өжү“ејҖжҠҳеҸ гҖӮ

<summary>ж Үзӯҫз”ЁжқҘе®ҡеҲ¶жҠҳеҸ еҶ…е®№зҡ„ж ҮйўҳгҖӮ

<details>
  <summary>иҝҷжҳҜж Үйўҳ</summary>
  иҝҷжҳҜдёҖж®өи§ЈйҮҠж–Үжң¬гҖӮ
</details>

дёҠйқўзҡ„д»Јз ҒжҳҫзӨәз»“жһңеҰӮдёӢгҖӮ

в–¶ иҝҷжҳҜж Үйўҳ

зӮ№еҮ»еҗҺпјҢеұ•зӨәзҡ„ж•ҲжһңеҰӮдёӢгҖӮ

в–ј иҝҷжҳҜж Үйўҳ
иҝҷжҳҜдёҖж®өи§ЈйҮҠж–Үжң¬гҖӮ

йҖҡиҝҮ CSS и®ҫзҪ®summary::-webkit-details-markerпјҢеҸҜд»Ҙж”№еҸҳж ҮйўҳеүҚйқўзҡ„дёүи§’з®ӯеӨҙгҖӮ

summary::-webkit-details-marker {
  background: url(https://example.com/foo.svg);
  color: transparent;
}

дёӢйқўзҡ„ж ·ејҸжҳҜеҸҰдёҖз§ҚжӣҝжҚўз®ӯеӨҙзҡ„ж–№жі•гҖӮ

summary::-webkit-details-marker {
  display: none;
}
summary:before {
  content: "\2714";
  color: #696f7c;
  margin-right: 5px;
}

JavaScript APIвҖӢ

Detailsе…ғзҙ зҡ„openеұһжҖ§иҝ”еӣһ<details>еҪ“еүҚжҳҜжү“ејҖиҝҳжҳҜе…ій—ӯгҖӮ

const details = document.querySelector('details');

if (detail.open === true) {
  // еұ•ејҖзҠ¶жҖҒ
} else {
  // жҠҳеҸ зҠ¶жҖҒ
}

Detailsе…ғзҙ жңүдёҖдёӘtoggleдәӢд»¶пјҢжү“ејҖжҲ–е…ій—ӯжҠҳеҸ ж—¶пјҢйғҪдјҡи§ҰеҸ‘иҝҷдёӘдәӢд»¶гҖӮ

details.addEventListener('toggle', event => {
  if (details.open) {
    /* еұ•ејҖзҠ¶еҶө */
  } else {
    /* жҠҳеҸ зҠ¶жҖҒ */
  }
});
