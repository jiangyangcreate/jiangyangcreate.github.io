## 多媒体

你可以从[emoji-cheat-sheet](https://github.com/ikatyang/emoji-cheat-sheet)中找到支持的表情。


### 图片

```markdown
![图片描述](图片地址)
```

### 音视频

`markdown`中使用`html`标签插入音视频。

```markdown
<audio src="语音地址"></audio>

<video src="视频地址"></video>
```

## 结构化

### 标题

```markdown

### 二级标题

#### 三级标题
```

```markdown
* 学习使用*号创建无序列表
```
### 任务列表

- [x] 任务 1
- [ ] 任务 2

```markdown
- [x] 任务 1
- [ ] 任务 2
```
