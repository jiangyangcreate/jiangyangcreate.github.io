---

一提到图表，大家脑海里浮现的，通常是柱状图、饼图、趋势图等等。这是按照图形等维度对图表进行分类，经常会导致图表的误用。

图表的作用，是帮助我们更好地看懂数据。选择什么图表，需要回答的首要问题是『我有什么数据，需要用图表做什么』，而不是 『图表长成什么样』 。


推荐绘图库：
- [matplotlib](https://matplotlib.org/)
- [pyecharts](https://github.com/pyecharts/pyecharts)
