Python 标准库非常庞大，合并子模块仍有一百四十多个库和二十多个类别。涉及范围广泛，库包含了以 C 编写和以 Python 编写的模块，有些模块经过专门设计，通过将特定平台功能抽象化为平台中立的 API 来鼓励和加强 Python 程序的可移植性。

这里挑选了最值得深入掌握的标准库模块，按主题分类组织。未被提及的标准库可以前往 [Python 标准库](https://docs.python.org/zh-cn/3.14/library/index.html) 浏览完整内容。

:::tip
有些模块是新手入门最先接触的模块，且相当常用，例如：`time`、`random`。

但其并非语言运行的核心模块，此部分模块放在**按需查阅**部分。建议快速浏览，有需要时再深入学习。
:::

Python 三方库数量极其巨大，[awesome-python](https://awesome-python.com/) 是一份带有观点的优秀 Python 框架、库、软件和资源清单。你可以根据此清单快速了解Python三方库的生态圈。
