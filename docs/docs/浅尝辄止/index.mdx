本部分内容是个人摸索技术成长方向的过程中的产物。

| 技术方向 | 学习/应用原因 | 停止/局限原因 |
| --- | --- | --- |
| **Html & JavaScript** | 网页开发基础。 | 若工作不涉及前端，缺乏实践场景，极易遗忘。 |
| **网页自动化** | 在办公自动化、中小型爬虫领域具有明显优势。 | 深度应用场景有限，难以支撑更复杂的业务需求。 |
| **数据分析** | 做 AI 应用时会有数据预处理、简单统计等需求。 | 不必走「数据分析师」那条深线，会用常见操作（读表、过滤、简单统计）即可。 |
| **深度学习** | 了解AI基础知识，知晓调用逻辑。 | 调用成熟的 API（OpenAI/Claude）来构建自动化流，比去研究 Transformer 底层数学公式更符合你的效率原则。 |
| **项目管理** | 提升宏观把控能力。 | 现阶段仅需“了解”即可，无需深入钻研。 |
| **Nginx** | 传统的反向代理与网关配置。 | 配置后基本不动，且现代开发多基于 Docker 或 Serverless，运维工作已被工具高度抽象化。 |
